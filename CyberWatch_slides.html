Cyber Watch — Emotion-aware Threat Detection
Multimodal detection (Text | Face | Voice)
Prototype demo: webcam + microphone + files
Problem Statement
  • Multimodal threats: phishing, aggressive calls, suspicious video
  • Current tools miss emotional/context cues
  • Human moderators face high-volume multimedia
Objectives
  • Modular prototype for text/voice/face
  • Usable GUI with logging
  • Lightweight fallbacks and heavy-model support
  • Real-time monitoring & alerting
Methodology
  • Text: RoBERTa + heuristics
  • Voice: MFCCs + transformer ASR/emotion or RandomForest fallback
  • Face: YOLO / DeepFace / Haar cascade fallbacks
System Design
  • Frontend: main UI (Tkinter)
  • Models: voice_model.py, text_model.py
  • Utilities: file utils, DB history
Implementation Highlights
  • Robust GUI with theme and tooltips
  • VoiceThreatClassifier supports transformer & fallback
  • FacialEmotionAnalyzer uses multiple detection strategies
Demo & Results
  • Demo steps: voice file, webcam face analyzer, text threat popup
  • Results: file-based tests passed; alerts logged
Conclusion & Next Steps
  • Prototype validates multimodal detection
  • Next: CI, dataset, UX polish, dockerized demo
